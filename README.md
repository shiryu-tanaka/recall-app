# Recall アクティブリコール学習アプリ

間隔反復学習を活用した問題管理・学習支援Webアプリケーションです。
[(recall-app-alpha.vercel.app)](https://recall-app-alpha.vercel.app/)

## アプリ概要

Recall Appは、アクティブリコールという「記憶から能動的に思い出す作業」を繰り返す作業を支援するアプリケーションです。アクティブリコールは科学的に最も効果的といわれる学習方法です。

### 主な特徴

- 問題の作成・編集・削除
- カテゴリ機能による問題の整理
- 自動タスク生成（1日後、3日後、1週間後、2週間後、1ヶ月後）
- 今日のタスク・週間タスクの表示
- 正解・不正解による進捗管理
- ユーザー認証機能

## なぜ作ったか
このアプリを作った主な目的は、アウトプットと復讐のハードルを下げて、私自身の学習効率を上げるためです。
私は学ぶこと自体は好きなのですが、勉強してインプットするだけで満足する節があり、アウトプットや復讐がおろそかになりがちでした。
そのような問題を解決するため、自分で問題を作成し、自動で復習スケジュールを組んでくれるアプリが欲しいと思い開発しました。

### 間隔反復スケジュール

問題作成時に以下のスケジュールで自動的にタスクが生成されます：
- 1日後
- 3日後
- 1週間後
- 2週間後
- 1ヶ月後
